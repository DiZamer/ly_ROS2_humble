!На компьютере должен быть установлен docker desktop, wsl через командную строку (в cmd ввести wsl --update) и нужные пакеты для VSCode (приложен архив). Также
должны быть включены: виртуализация в BIOS, Hyper-V в панели управления (потребуется перезагрузка компьютера).

Скачать проект по ссылке https://github.com/DiZamer/ly_ROS2_humble и распокавать в любом удобном месте. Открыть папку в VSCode, 
нажать Reopen in container, дождаться загрузки (перестанут появляться новые команды в логах). Открыть новый терминал и в нем выполнять команды:
cd /ly_ROS2_humble/turtlebot3_ws
rm -rf src/ build/ install/ log/
mkdir src && cd src
git clone -b humble https://github.com/ROBOTIS-GIT/DynamixelSDK.git
git clone -b humble https://github.com/ROBOTIS-GIT/turtlebot3_msgs.git
git clone -b humble https://github.com/ROBOTIS-GIT/turtlebot3.git
git clone -b humble https://github.com/ROBOTIS-GIT/turtlebot3_simulations.git

cd ..
source /opt/ros/humble/setup.bash
colcon build --symlink-install

source install/setup.bash
export TURTLEBOT3_MODEL=waffle_pi
ros2 launch turtlebot3_gazebo turtlebot3_world.launch.py
